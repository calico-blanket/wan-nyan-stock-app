# 🐾 ワンニャン株式予報

日本株のスイングトレードを支援する、Google Apps Script 製の AI 自動分析システムです。
5体のAIキャラクターが毎日自動でデータを収集・分析し、Web アプリでレポートを届けます。

## ✨ 特徴

- **5体のAIキャラクター**がそれぞれ異なる分析を担当
- **Yahoo Finance 非公式API** で日足データを自動取得
- **テクニカル指標を自動計算**（ボリンジャーバンド・RCI・MACD・移動平均線配列）
- **毎日自動実行**（Google Apps Script タイムトリガー）
- **Web アプリ**でいつでもレポートを確認（スマホのホーム画面に登録可能）
- **完全無料**で運用可能（Google アカウント + Gemini API 無料枠）

## 🚀 セットアップ手順

### 必要なもの

- Google アカウント
- Gemini API キー（[Google AI Studio](https://aistudio.google.com/app/apikey) で無料取得）

### 手順

#### Step 1：Google Apps Script の準備

1. [Google スプレッドシート](https://sheets.google.com) で新規スプレッドシートを作成
2. メニュー「拡張機能」→「Apps Script」を開く
3. `wan_nyan_stock.gs` の内容をエディタに貼り付けて保存

#### Step 2：Gemini API キーを設定

1. GAS エディタのメニューから「株式予報アプリ」→「Gemini API キーを設定」
2. Google AI Studio で取得したキーを貼り付けて OK

#### Step 3：初期セットアップ

1. 「株式予報アプリ」→「① 初期セットアップ」を実行
2. 各種シートが自動作成されます

#### Step 4：Web アプリとしてデプロイ

1. GAS エディタ右上「デプロイ」→「新しいデプロイ」
2. 種類：「ウェブアプリ」を選択
3. 次のユーザーとして実行：**自分**
4. アクセスできるユーザー：**全員**
5. 「デプロイ」をクリックし、表示された URL を控える

#### Step 5：トリガーを設定

1. 「株式予報アプリ」→「トリガーを一括設定（17:00/17:30/18:00）」を実行
2. 必要に応じて「週次トリガーを設定」も実行
3. 権限承認ダイアログが出たら「承認」をクリック

## ⚙️ 設定のカスタマイズ

コード冒頭の定数で各種設定を変更できます。

```javascript
var SCREEN_TOP_N = 8;       // スクリーニングで抽出する銘柄数
var YAHOO_RANGE  = '6mo';   // Yahoo Finance から取得する期間
var BB_PERIOD    = 20;      // ボリンジャーバンドの期間
var RCI_SHORT    = 9;       // RCI 短期
var RCI_LONG     = 26;      // RCI 長期
```

監視銘柄の追加・削除は `pool` シートを直接編集するだけで反映されます。

## ⚠️ 注意事項・免責事項

- Yahoo Finance の**非公式エンドポイント**を利用しているため、仕様変更により突然動作しなくなる可能性があります
- Gemini API 無料枠には **1日あたりのリクエスト数制限**があります（目安：リセットは日本時間夕方ごろ）
- GAS には **1日あたりの実行時間制限**（無料：90分/日）があります
- 株価データはリアルタイムではなく、**一定の遅延**があります
- **本システムは情報提供を目的としており、投資を推奨・勧誘するものではありません**
- AI の分析結果は必ずしも正確ではなく、**投資損失について一切の責任を負いません**
- **投資はご自身の判断と責任において行ってください**
