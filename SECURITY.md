# セキュリティについて

## APIキーの取り扱い

本プロジェクトでは、Gemini API キーを **Google Apps Script のスクリプトプロパティ** に保存します。
コード中に直接 API キーを記述しないでください。

### 安全な設定方法

スプレッドシートのメニュー「株式予報アプリ」→「Gemini API キーを設定」から登録してください。
キーはスクリプトプロパティに保存され、コードには一切含まれません。

### GitHub にプッシュする前に確認すること

コード冒頭の以下の行が `'YOUR_GEMINI_API_KEY'` のままになっているか確認してください。

```javascript
var GEMINI_API_KEY = 'YOUR_GEMINI_API_KEY';
```

実際のキーが入った状態でコミット・プッシュしないようにご注意ください。

## 脆弱性の報告

本プロジェクトに関するセキュリティ上の問題を発見された場合は、
Issue ではなく非公開の手段でご連絡ください。
